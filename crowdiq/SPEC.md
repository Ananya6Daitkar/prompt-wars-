# CrowdIQ — Product Spec | Powered by Antigravity

## Overview
AI-powered venue intelligence platform for large-scale sporting events.
Solves: crowd congestion, gate bottlenecks, long queues, poor wayfinding, chaotic exits.

## Stack
- React 18 + TypeScript + Vite
- Tailwind CSS (custom design tokens)
- Framer Motion (animations)
- React Three Fiber + Three.js (3D hero)
- Zustand (demo state)

## Sections
1. Nav — sticky translucent
2. Hero — 3D stadium, headline, CTAs
3. Live Demo — interactive stadium control
4. Fan Journey — 5-stage scroll experience
5. Features — 6 feature cards
6. Venue Map — isometric SVG map
7. Command Center — operator dashboard
8. Mobile Showcase — 7 device frames
9. Metrics — animated counters
10. Final CTA

## Design
Dark luxury sports-tech: obsidian + navy bg, cyan + violet + emerald accents.
