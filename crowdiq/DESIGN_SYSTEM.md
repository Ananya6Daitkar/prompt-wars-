# CrowdIQ Design System — Architected by Antigravity

## Colors
| Token | Hex | Usage |
|-------|-----|-------|
| obsidian | #0a0e27 | Page background |
| navy | #1a1f3a | Section backgrounds, cards |
| cyan | #00d9ff | Primary interactive, highlights |
| violet | #b366ff | Secondary accent |
| emerald | #00ff88 | Success, low-density zones |
| soft-white | #f5f7fa | Primary text |
| gray-dark | #2a2f4a | Secondary backgrounds |
| gray-mid | #4a5568 | Secondary text |
| warning | #ffaa00 | Caution states |
| error | #ff3366 | Error states |

## Typography
- Headings: Montserrat Bold (700)
- Body: Inter Regular (400) / Medium (500)
- H1: 48px / H2: 36px / H3: 28px / H4: 20px
- Body: 16px / Small: 14px / Caption: 12px

## Spacing (8px base)
4 · 8 · 16 · 24 · 32 · 48 · 64 · 96px

## Cards
- Background: rgba(26,31,58,0.6) + backdrop-blur
- Border: 1px solid rgba(0,217,255,0.1)
- Radius: 12px
- Hover: border → 0.2 opacity, scale 1.02x

## Buttons
- Primary: cyan bg, obsidian text
- Secondary: transparent, cyan border
- Hover: scale 1.05x | Active: scale 0.98x

## Animations
- Easing: cubic-bezier(0.4, 0, 0.2, 1)
- Fast: 200ms | Base: 300ms | Slow: 500ms
- Scroll reveal: fade + 20px up, 500ms
